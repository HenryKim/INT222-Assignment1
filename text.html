Basic html text

Text is the most important feature of HTML because without it, we can not understand anything. There are many tags that can display texts in which it can either be indented or by printing it to a new line.

HTML tags that can be used with text:

  • <p>........</p>
  • <h1>........</h1>
  • <h2>........</h2>
  • <h3>........</h3>
  • <h4>........</h4>
  • <h5>........</h5>
  • <h6>........</h6>
  • <pre>.......</pre>

Text with CSS - Example #1

"When one door closes another opens - but often we look so long so regretfully upon the closed door that we fail to see the one that has opened for us."

The CSS feature that is used in this example is the text-decoration feature. The text-decoration feature has been set to underline so the text prints with an underline.

Text with CSS - Example #2

"There are two things to aim at in life; first to get what you want, and after that to enjoy it. Only the wisest of mankind has achieved the second."

The CSS feature that is used in this example is the font-style feature. The font-style feature has been set to italic so the text prints with an italic style font.

Text with CSS - Example #4

"All compromise is based on give and take, but there can be no give and take on fundamentals. Any compromise on mere fundamentals is surrender - for it is all give and no take."

The CSS features that are used in this example are the text-decoration feature, colour feature, and the font-style feature. The text-decoration feature has been set to underline so the text prints with an underline; the colour feature has been set to #008000 so the text prints in green colour, and The font-style feature has been set to segrif so the text prints with a segrif style font.

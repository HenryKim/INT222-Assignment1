html Tables

Basic html table

Tables are one of the most useful feature in HTML. Tables are often being used to represent and organize data.Multiple rows and columns can be created using different table tags. Table descriptions can also be applied using the caption tag.


HTML Tags that can be used in the table tag:

  • <caption>.....</caption>
  • <tr>..........</tr>
  • <th>..........</th>
  • <td>..........</td>

Table with CSS - Example #2

Table with Cell Spacing
Column Heading 1 Column Heading 2 Column Heading 3
Row 2, Column 1 Row 2, Column 2 Row 2, Column 3
Row 3, Column 1 Row 3, Column 2 Row 3, Column 3
Row 4, Column 1 Row 4, Column 2 Row 4, Column 3

The CSS features that are used to make the table look better are the width, border, margin, cell border, background colour for the cells, and border-spacing. The width is set to 400 pixels. The border for the table is set to 4 pixels. The margin is set to auto so the table appears at the center. The border pixel for the cells is set to 2 pixels, and the of #0000ff is set to the background colour for the heading which makes the 1st row of columns to look blue. Border-spacing is set to 20 pixels which makes each cell to look like seperate rectangles inside a big rectangle.

The CSS features that are used to make the table look better are the width, border, margin, cell border, rowspan, and colspan. The width is set to 600 pixels. The border for the table is set to 4 pixels. The margin is set to auto so the table appears at the center. The border pixel for the cells is set to 2 pixels. Colspan is set to 2 which makes the heading to overlap to the 2nd column. Rowspan is set to 1 which makes the text overlaps row 1 and row 2.

Computer Related Terms:


Bug
It is a term used when a programming code is not stable or a part of the code which makes the program not to work properly.
DNS (Domain Name System)
As we all know, different websites have its own ip address. The DNS translates these ip addresses into words so people can easily remember the web page address.
Encryption
It is a type of security feature which protects the contents of a file by asking the user for a password to decrypt the file
HTML (Hypertext Markup Language
A programming language that is used to make web pages. It is also known as the most required programming language to make web sites.
Virus
A computer code which harms the operating system. It can potentially damage the files associated with the operating system
